# GitHub formation syntax

> This Repository should introduce you to the Github formation syntax.





## Headings

To create a heading in your file you can add the following syntax before the text `#`. You can also put multiple `#` infront of the text, the maximum is 6.


Code:

```
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6
```

## Styling text

There are several ways to customize your text.



### Bold


#### Syntax

`** **` or `__ __`

#### Example

`**This is a bold text**`

or

`__This is a bold text__`

#### Output

**This is a bold text**



### Italic


#### Syntax

`* *` or `_ _`

#### Example

`*This is a italic text*`

or

`_This is a italic text_`

#### Output

*This is a bold text*



### Strikethrough


#### Syntax

`~~ ~~`

#### Example

`~~This is a strikethrough text~~`

#### Output

~~This is a strikethrough text~~



### Subscript


#### Syntax

`<sub> </sub>`

#### Example

`<sub>This is a subscript text</sub>`

#### Output

<sub>This is a subscript text</sub>



### Superscript


#### Syntax

`<sup> </sup>`

#### Example

`<sup>This is a superscript text</sup>`

#### Output

<sup>This is a superscript text</sup>
